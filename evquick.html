eVQUICK

eVQUICK is an Android app development project that aims to provide users with real-time information about the nearest electric vehicle charging stations. The app allows users to view the estimated waiting time at a particular station, helping them make informed decisions about where to charge their vehicles.

Tools Used

  • Android Studio: Integrated development environment for Android app development.
  • Java: Programming language used for developing the app.
  • Gradle: Build automation tool used for managing dependencies and building the app.
  • GitHub: Platform for version control and collaboration.
  • Google Firebase: Backend-as-a-Service for user authentication and data storage.
  • Google APIs: Provides access to various Google services and data.
