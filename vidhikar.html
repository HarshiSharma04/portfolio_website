VIDHIKAR - AI Voice Assistant

VIDHIKAR is an advanced AI voice assistant designed to enhance productivity and convenience. It provides a wide range of functionalities, including sending emails, setting alarms, making web searches, playing music, and even built-in games. Developed with Python and various Python libraries, VIDHIKAR leverages cutting-edge technology to offer a seamless user experience.

Key Features

  • Send Emails: Compose and send emails through voice commands.
  • Set Alarms: Easily set and manage alarms using voice input.
  • Web Searches: Perform web searches hands-free with voice queries.
  • Play Music: Control music playback and discover new tracks by voice.
  • Built-in Games: Enjoy a variety of entertaining games within the assistant.

Tech Stack

  • Programming Language: Python
  • Libraries: Various Python libraries for natural language processing, voice recognition, and more.
  • Voice Recognition: Utilizes libraries like `SpeechRecognition` for accurate voice input.
  • Text-to-Speech: Implements libraries such as `pyttsx3` for converting text to speech.
  • Email Handling: Uses `smtplib` for sending emails.
  • Web Scraping: Employs `BeautifulSoup` and `requests` for web data retrieval.
  • Game Integration: Includes simple games implemented with Python libraries.
