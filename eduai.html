EduAI

A user-centric smart learning application with features like a personalized learning style predictor, AI Tutor, recommendation system, and focus mode.

Features

Learning Style Predictor

The learning style predictor, powered by a Random Forest machine learning model, determines the user's learning style from options like Visual, Auditory, Reading/Writing, and Kinesthetic.

Recommendation System

Our recommendation model combines three ML models to provide accurate course and study material recommendations based on the user's preferred subjects and difficulty levels.

AI Tutor

The built-in AI chatbot serves as a tutor, answering user queries and preventing the need to switch to other sources for learning or doubt resolution.

Focus Mode

A timer feature that helps users track their focused study time, ensuring productivity and effective learning sessions.

Tech Stack

  • Frontend: Flutter for Android development
  • Backend: Python, Flask for server-side development
  • Data Processing: Pandas, Sklearn for model development and integration
  • APIs: YouTube API and Chatbot API integrated
  • Plugins: Various plugins for enhanced app features
